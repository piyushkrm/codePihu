Rock Paper Scissors Game

Play the classic game against the computer!

Objective

The goal of this Rock-Paper-Scissors game is to allow users to play the popular hand game against the computer. The user can choose between rock, paper, or scissors, and the game determines the winner based on traditional rules: rock beats scissors, scissors beats paper, and paper beats rock.

Overview

This game is built with HTML, CSS, and JavaScript, providing a simple and interactive interface for users. It allows players to choose their move, displays the computer's move, and shows the result after each round.

Key Features

  • Simple and easy-to-use interface.
  • Randomly generated computer choices.
  • Displays the result after each round (win/lose/draw).
  • Reset button to restart the game.

Technology Stack

  • Frontend: HTML, CSS, JavaScript
  • Libraries/Tools: None (pure HTML, CSS, and JavaScript)
