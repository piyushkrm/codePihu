Covid Infected Map

Visualizing COVID-19 cases across the globe in real-time.

Objective

The Covid Infected Map is an interactive tool to track and display the spread of COVID-19 globally or regionally using a geographic map.

Overview

This project uses a map visualization library to represent COVID-19 infection data. Users can interact with the map to explore case details in specific regions.
