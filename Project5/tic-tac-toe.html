Tic-Tac-Toe Game

Play the classic 3x3 grid game against a friend or the computer!

Objective

The objective of the game is to get three of your marks (X or O) in a row (horizontally, vertically, or diagonally) on a 3x3 grid before your opponent does.

Overview

This Tic-Tac-Toe game is built using HTML, CSS, and JavaScript. Players can play against each other or against the computer. It checks for a winner and allows the players to restart the game once it's over.

Key Features

  • 2-player mode or play against the computer.
  • Automatic winner detection after every move.
  • Responsive UI that adjusts for different screen sizes.
  • Reset button to start a new game.

Technology Stack

  • Frontend: HTML, CSS, JavaScript
  • Libraries/Tools: None (pure HTML, CSS, and JavaScript)

Working Process

  1. Grid Layout: Create a 3x3 grid for the game board.
  2. Player Movement: Allow the player to select a square to place either an "X" or an "O".
  3. Winner Detection: After each move, check for a winner or a draw.
  4. Game Reset: Implement a reset function to restart the game once it's finished.

Future Enhancements

  • Implement AI for single-player mode with difficulty levels.
  • Add an option for players to customize their marks (X/O).
  • Enable a scoreboard to track multiple games.
